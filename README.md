Software for practising simplex


Notes:
- To type a value as a result of dividing by 0, type "undefined" or "inf"
- Can only do simplex with <= inequalities at the moment but maybe I'll change this later
- Type reset to start a new table
- Press show answer to show answers (you can go back to the original table by clicking hide answers after doing this).
- If you want to not do a table, press show answers then next to move onto the next table.
- You have to type the initial table thing out twice - once when entering and the next on the first iteration but if you want to skip doing this, just press show answer for the first table thing then move onto the next. 

Let me know if you find any bugs / anything should be changed .
